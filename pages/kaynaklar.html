Kaynaklar & Linkler

Bu kursun ötesinde Unity öğrenmeye devam etmek için en güvenilir resmi kaynaklar, topluluk YouTube kanalları, asset siteleri, kitaplar ve forumlar.

📚 Resmi Kaynaklar

Unity Documentation

Resmi belgeler — Manual + Scripting API her konunun referans kaynağı.

docs.unity3d.com

Unity Learn

Resmi ücretsiz öğrenme platformu — pathway'ler (Junior Programmer, Creative Core), microgame'ler.

learn.unity.com

Unity YouTube

Resmi YouTube — tutorial, livestream, Unite konferans sunumları.

youtube.com/@unity

Unity Asset Store

Ücretsiz/ücretli asset, paket, eklenti, örnek proje (URP Boat Attack, FPS Microgame).

assetstore.unity.com

Unity GitHub

Açık kaynak Unity örnek projeleri: 2D Game Kit, BoatAttack (URP), ECS samples, Animation Rigging.

github.com/Unity-Technologies

Unity Discussions

Resmi forum — soru sor, eski tartışmaları ara, Unity ekibi de yanıtlar.

discussions.unity.com

🎥 Topluluk YouTube Kanalları

Code Monkey

Hugo Pereira — pratik tutorial'lar, "Kitchen Chaos" kapsamlı kursu ücretsiz. Yeni başlayanlar için ideal.

youtube.com/@CodeMonkeyUnity

Tarodev

İleri seviye, mimari odaklı — ScriptableObject mimarisi, kod organizasyonu, profesyonel pratikler.

youtube.com/@Tarodev

Brackeys (arşiv)

Klasik. Tutorial'lar 2020'de durdu ama eğitim değeri hâlâ yüksek — yeni başlayan için altın.

youtube.com/@Brackeys

Sebastian Lague

Sanat eseri seviyesinde — prosedürel jenerasyon, AI, görsel matematik. "Coding Adventures" serisi efsane.

youtube.com/@SebastianLague

Freya Holmér

Shader Forge yaratıcısı. Matematik ve shader öğretiminde efsanevi görsel açıklamalar.

youtube.com/@FreyaHolmer

Game Maker's Toolkit

Mark Brown — motor-bağımsız oyun tasarımı. Bilmeyi her gamedev'in zorunlu olduğu kanal.

youtube.com/@GameMakersToolkit

Jason Weimann

Mimari, design pattern, "Unity Architecture for Noobs" gibi profesyonel pratik dizileri.

youtube.com/@jasonweimann

Catlike Coding

Yazılı, derin teknik. Shader, render pipeline, ECS, prosedürel mesh — akademik tat.

catlikecoding.com/unity/tutorials

📖 Önerilen Kitaplar

Game Programming Patterns

Robert Nystrom — ücretsiz online. Singleton, Observer, State, Command pattern'leri öğrenmek için altın metin.

gameprogrammingpatterns.com

Unity in Action (3rd ed.)

Joseph Hocking — klasik referans, 3B / 2B oyun projeleri üzerinden Unity öğrenimi.

manning.com

Game Engine Architecture

Jason Gregory — motor iç mekaniğini anlamak için (ileri seviye, AAA mühendislik).

gameenginebook.com

The Art of Game Design

Jesse Schell — tasarım odaklı, "lens" kavramı. Her gamedev'in okuması gereken klasik.

schellgames.com

Clean Code & The Pragmatic Programmer

Robert C. Martin / Andrew Hunt — Unity-spesifik değil ama C# alışkanlıkları için zorunlu okumalar.

amazon.com

The Book of Shaders

Patricio Gonzalez Vivo — ücretsiz online. Shader matematiğine en görsel giriş.

thebookofshaders.com
